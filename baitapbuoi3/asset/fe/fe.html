Khóa Học FRONT END & REACTJS

Học lập trình Web Frontend + React 5 tháng 60 buổi với hình thức đào tạo Offline hoặc 7 tháng 84 buổi với hình thức đào tạo Online. Cam kết việc làm cho học viên sau khi tốt nghiệp với cả hai hình thức đào tạo. Đội ngũ Giảng viên dày dặn kinh nghiệm, Giáo trình đào tạo bài bản, liên tục cập nhật các kiến thức mới nhất, bám sát thực tế khi làm việc.

Đối Tượng Hoc Viên

  1. Những người đang thất nghiệp hoặc công việc bấp bênh, muốn có công việc ổn định, lương cao và không bị ảnh hưởng bởi COVID-19.
  2. Sinh viên, người đi làm trái ngành muốn trở thành lập trình viên web.
  3. Sinh viên CNTT năm cuối làm đồ án tốt nghiệp web front end.
  4. Nhân viên marketing, thiết kế đồ hoạ muốn bổ sung kỹ năng lập trình web hoặc phát triển thành UI/UX designer.

Nội Dung Chính

  1. Thiết kế giao diện UI và phân tích , tối ưu trải nghiệm người dùng.
  2. Cắt giao diện từ file PSD, Sketch, Figma.
  3. Lập trình JavaScript, jQuery kết hợp CSS, Bootstrap để thiết kế responsive web.
  4. Lập trình JavaScript ES6.
  5. Lập trình ứng dụng Single Page App với ReactJS.
  6. Hoàn thành 3 đồ án môn học: thiết kế giao diện, lập trình web site hoàn chỉnh và ứng dụng Single Page App với ReactJS.
